# UI 设计软件更新日志的爬取

## 目录结构
```
├── README.md 
├── clock.py // 定时爬取代码
├── figma.py // figma 的爬取方法
├── figma.txt // figma 爬取到的内容
├── pixso.py // pixso 的爬取方法
├── pixsoLog.json // pixso 爬取到的内容
└── rss2json // 将现有的 rss 格式文件转为 json 格式
    ├── figma.json // 转换完的数据
    ├── index.js // 转换代码
    └── package.json
```
